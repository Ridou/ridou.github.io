1. Padding is the area between the content and the border. The margin is transparent and is the outer section of the content. 2. The id element should be used when it is more specific object. Most of the time it is best practice to use classes. 3. static is the default choice for every element. Relative positioning depends on having an attribute to be positioned to, for example bottom 15px would shift it 15px up from its original position. Absolute positioning has the element placed relative to the first positioned parent element. Fixed positioning is relative to the browser window. 4.The block elements will fit their parent containers unless a width or height are specified and it ignores the vertical-align property. Inline elements will ignore width and height properties and if floated left or right it will automatically become a block level element. It is usually a box that acts just like text.
