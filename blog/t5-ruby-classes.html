Ruby Class

Class of Ruby

A class is defined with a class statement and contains methods to work on objects, which are defined with a def and with an end. A method called initialize is the constructor which defines how you create an object.

There can be instance variables that are differentiated with a @ symbol at the start of the name of the variable. These variables are applied to the current object. There are also global variables that can be used through out the program.

Ruby classes make ruby much more dynamic when working through objects and it allows to further group this object oriented language. The more I practice using these classes and its methods inside of them, the more I can manipulate the objects.
-Richard Santin
